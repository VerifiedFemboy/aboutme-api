## How to use it?

1. Install [rustup](https://www.rust-lang.org/tools/install)
2. Clone this repository by using git (command: git clone https://github.com/VerifiedFemboy/aboutme-api.git)
3. cd aboutme-api
4. Run this API (command: cargo run)
5. Enjoy ;3
